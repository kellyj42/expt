How to Use EduResource Hub

1. Getting Started

1.1 Create an Account

Visit the EduResource Hub website and click on "Sign Up" to create a new account. Enter your details (email or student ID) and create a password. Verify your email address to activate your account.

1.2 Log In

Go to the login page and enter your email/student ID and password. If you forget your password, use the "Forgot Password" link to reset it.

2. Accessing Resources

2.1 Search for Resources

Use the search bar to find resources by subject, course, year, or resource type (e.g., past papers, videos, audio files). Apply filters to sort resources by most downloaded, recent, or highest-rated.

2.2 Upload Resources

Click on "Upload" to share educational resources with the community. Select the file (PDF, MP4, MP3, etc.) and add details like subject, course, and year. Ensure the file size is within the limit (e.g., 100MB).

2.3 Download Resources

Click on a resource to view its details. Click "Download" to save the file to your device. For offline access, download resources and access them later without an internet connection.

3. Participating in Discussions

3.1 Join the Forum

Go to the Discussion Forum from the main menu. Browse through topics or search for specific discussions.

3.2 Ask Questions or Share Insights

Click on "New Topic" to start a discussion. Reply to existing threads to share your thoughts or ask follow-up questions.

3.3 Follow Discussions

Click the "Follow" button on a thread to receive notifications about new replies.

5. Notifications

5.1 Stay Updated

Receive notifications about new resources, upcoming exams, or forum replies. Notifications can be viewed in the app or sent to your email.

6. Admin Features (For Admins Only)

6.1 Manage Users

Access the Admin Dashboard to manage user accounts, roles, and permissions.

6.2 Manage Resources

Review and approve uploaded resources. Remove outdated or inappropriate content.

6.3 Monitor Analytics

Track resource downloads, user activity, and system performance using the analytics dashboard.

7. Mobile Access

EduResource Hub is fully responsive and works on both desktop and mobile devices. Download the mobile app (if available) for a smoother experience.
